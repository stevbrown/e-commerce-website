<h2>Описание проекта</h2>
Этот проект представляет собой реализацию веб‑страницы по макету из Figma, выполненную в рамках учебной практики. Цель проекта — отработать навыки вёрстки, работы с дизайн‑макетами и применения современных веб‑технологий.

<h3>Технические характеристики</h3>
- **HTML5** — структура страницы;
- **CSS3** (с использованием Flexbox/Grid) — стилизация и адаптивная вёрстка;
- **JavaScript** — интерактивные элементы;
- **Figma** — исходный дизайн‑макет;

<h3>Как запустить проект</h3>
1. Скопируйте репозиторий на свой компьютер:
   ``git clone <ссылка-на-репозиторий>``
2. Перейдите в папку проекта:
   ``cd project-name``
3. Откройте файл ``index.html`` в любом современном браузере.
