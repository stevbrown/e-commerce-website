<h2>Описание проекта</h2>
Этот проект представляет собой реализацию веб‑страницы по макету из Figma, выполненную в рамках учебной практики. Цель проекта — отработать навыки вёрстки, работы с дизайн‑макетами и применения современных веб‑технологий.

<h3>Технические характеристики</h3>
<ul>
   <li><b>HTML5</b> — структура страницы;</li>
   <li><b>CSS3</b> (с использованием Flexbox/Grid) — стилизация и адаптивная вёрстка;</li>
   <li><b>JavaScript</b> — интерактивные элементы</li>
   <li><b>Figma</b> — исходный дизайн‑макет</li>
</ul>

<h3>Как запустить проект</h3>
<ol>
   <li>Скопируйте репозиторий на свой компьютер: <code>git clone <ссылка-на-репозиторий></code></li>
   <li>Перейдите в папку проекта: <code>cd <имя-проекта></code></li>
   <li>Откройте файл <code>index.html</code> в любом современном браузере.</li>
</ol>
или перейти на github-page сайт: https://stevbrown.github.io/e-commerce-website/
